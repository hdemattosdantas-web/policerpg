# Arquivos de Som

Esta pasta contém os efeitos sonoros do jogo Police RPG.

## Arquivos Necessários

- `radio-beep.mp3` - Som de bip quando recebe mensagem no rádio
- `success.mp3` - Som de sucesso em ocorrências
- `purchase.mp3` - Som de compra na loja
- `error.mp3` - Som de erro

## Formato Recomendado

- Formato: MP3
- Duração: 1-2 segundos
- Volume: Médio (não muito alto)
- Qualidade: 128kbps

## Como Adicionar

1. Coloque os arquivos de áudio nesta pasta
2. Os arquivos já estão configurados para serem usados pelo jogo
3. Os sons serão tocados automaticamente nos momentos apropriados

## Licença

Use sons livres ou licenciados para evitar problemas de direitos autorais.

Você pode usar qualquer som curto de rádio/beep ou gerar um online.
