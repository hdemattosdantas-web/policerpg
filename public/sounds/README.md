# Sons do Rádio

Adicione um arquivo chamado `radio-beep.mp3` nesta pasta para o som de "bip" do rádio.

Você pode usar qualquer som curto de rádio/beep ou gerar um online.

Exemplos de onde encontrar:
- https://freesound.org/people/juskiddink/sounds/173066/
- https://www.zapsplat.com/sound-effect-category/radio-beeps/

O arquivo deve ser chamado exatamente: `radio-beep.mp3`
